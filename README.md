# Internet-Speed-Test
PHPで作ったインターネット速度テストです。 
# 使い方 
お使いのレンタルサーバーまたはローカル開発環境にファイルを配置します。
`アドレス/設置したフォルダ/`の書式（例えば`localhost/speedtest/`）のURLにアクセスします。  
アクセスすると、直ぐにテストされます。
再テストしたい時は、再読み込みするか、「再測定」ボタンをクリックします。
